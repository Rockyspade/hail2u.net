The CSS Weblog

on Nov 3, 2004 under Web Design

ジャンルを絞ったblogをいろいろ提供している、Weblogs, Inc.でCSSをトピックにしたThe CSS Weblogというものが始まってたんですね。CSSのリソースをいろいろ集めて提示するというもので、系統的にはCSS VaultのResourcesの方と似た感じっぽいですが、こちらは「CSSって何よ？」とか「blogいじり始めて初めてCSS書いたかも！」みたいな人向けっぽい感じな。

ぶっちゃけ、ネタ的にはあんまり面白くないですね(ぶっちゃけすぎ)。JavaScriptと連携させるとかそんなお話のエントリとかもありますが、CSSハッカーの多くは「如何にCSSでどうにかするか」ということに執念を燃やしている人が多そう(偏見)なので、あんまりそういうのは興味ないんじゃないかなーとか。Pure CSSなハックとかもっとCSSの根源に迫る格調高い論述(なんだそれ)みたいなもの、すなわちCSSに特化したもののが良いなぁと。僕が知りたいだけですけど。
