ウェブデザインのためのドキュメント

on Jan 6, 2003 under Web Design

いのうさんへクリティカルにプレゼント。The Document Object Model in MozillaにZipped HTMLでMozillaのDOMリファレンスが置いてあります。さっきバイトに教えてもらったよ。いや、かなりイケてる。

Platform SDKの一部のInternet Development SDKがあれば、
HTML and DHTML Refernce(ちょっと怪しいリンクなのでお気をつけて)
が開けちゃったりします。内容的には、MSDN Library内の、
Web Development > HTML and Dynamic HTML > HTML and DHTML Refernce
と同一です。Internet Development SDKだけダウンロードすることも可能なので、とりあえずリファレンスが欲しかったりする人は、落としてみてはどうですか？オンラインのMSDN Libraryで閲覧するのは果てしなくダルいし。

他に、

  • HTML 4.01 Specification
  • CSS2 Specification
  • Core JavaScript Guide 1.5
  • Core JavaScript Reference 1.5

あたりもそろえてます。とほほもね。
