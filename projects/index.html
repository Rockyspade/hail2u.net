rsync-ac (Gruntプラグイン)

$ rsync -aC --exclude-from=.rsyncignore

rsync-ac (Gruntプラグイン)はrsyncを使って他のディレクトリへファイルをコピーするGruntプラグインです。オプション設定を絞ることであまり考えずに使えるようにしました。

pubsubhubbub_publish (Gruntプラグイン)

pubsubhubbub_publish (Gruntプラグイン)はGoogleが提供しているPubSubHubbubハブへ指定したURLのフィードをpublishするGruntプラグインです。

Picks

Picksは僕が興味をもった様々なURLをアイコンをヒントにざっと眺めることの出来るウェブサイトです。

UnUTM

UnUTMはウェブページのURLからGoogle Analyticsなどで利用されているUrchin Traffic Monitor (UTM)用のパラメーターを自動的に削除し、URLの分散を解消するJavaScriptライブラリです。

CSSWring

CSSWringはCSSを圧縮(minification)するNode.jsパッケージです。Node.jsパッケージとして組み込むだけでなく、コマンドラインからも実行できます。

CSS MQPacker

CSS MQPackerはCSSファイル内に散在するメディアクエリを、クエリごとにまとめCSSファイルを最適化するNode.jsパッケージです。これを利用したGruntプラグインもあります。

Long URL (Gruntプラグイン)

Long URL (Gruntプラグイン)はCSSファイルに含まれるURLの長さを調べ、2000文字以上のURLが見つかった場合に警告するGruntプラグインです。

Pit RO

Pit ROはPitのデータファイルをNode.jsで手軽に読み込めるようにするNodeパッケージです。

Selector4096 (Gruntプラグイン)

Selector4096 (Gruntプラグイン)はCSSファイルに含まれるセレクターの数を調べ、4096個以上あった場合に警告するGruntプラグインです。

G.F.

G.F.はGoogle Fontsで提供されているすべてのフォントをランダムな順番できれいなカラースキームを使って眺められるページです。

Detect Font Loading

Detect Font Loadingはフォントの読み込みの完了を検知し、html要素にクラス名を追加するJavaScriptライブラリです。主にChromeやSafari等でWebフォントの読み込みが完了するまでコンテンツが表示されない現象への対策に利用することを想定しています。

ロレム・イプサム

ロレム・イプサムはそこそこ意味の通った英数字交じりの日本語の文章をCC0で登録・収集するプロジェクトです。主にウェブ・ページやウェブ・アプリケーション等のモックアップに使うことを想定しています。

Sixty-Four

Sixty-Fourは新しい記事が出た時に必ず読むことにしている64のウェブログのリストです。

Drawic

DrawicはSVGで描かれたアイコンの小さなセットです。

Quote for the Japanese Typography

Quote for the Japanese Typographyは日本語タイポグラフィー周りのお話を引用し集めたページです。

Normalize.scss

Normalize.scssはNormalize.cssをSassで扱いやすいようにルールごとに切り分けてモジュラー化したSCSSライブラリです。

Missing Language Handlers for Google Code Prettify

Missing Language Handlers for Google Code PrettifyはGoogle Code Prettifyが対応していない言語のためのハンドラー・ファイルを提供します。

SCSS Column Equation

SCSS Column EquationはCSSでカラム・レイアウトを行う時に必要な面倒くさい計算をカラム幅とその隙間を定義するだけで代わりに行ってくれるSCSSライブラリです。

Hougan

Houganは方眼紙に似た実線と点線の組み合わせのグリッドに薄くノイズを乗せて紙らしくしたPNGのパターン画像です。

SCSS Partials

SCSS Partialsはよく使うclearfixや画像置換などのミックスインや有名なCSSライブラリをSassから簡単に扱えるようにSCSSライブラリとしてまとめたものです。

vim-css3-syntax

vim-css3-syntaxはVimのビルトイン・ランタイムに含まれるCSSのシンタックス・ファイルを拡張する形でCSS3の様々なプロパティーや機能に対応させるVimシンタックス・ファイルです。

H2U Colorscheme

H2U ColorschemeはgVim用のカラースキームのセットです。

Iine!

Iine!は興味を引いたロゴやテクスチャ、タイポグラフィーなどのWebページのウィジェットをコレクションしているWebサイトです。

MinA

MinAはJavaScriptのみで動作するAmazon.co.jpの全文検索ツールです。

Harmonize Text (jQueryプラグイン)

Harmonize Textは各行のフォントの大きさをその親要素の幅にちょうどおさまるように調整し、インパクトのあるヘッドラインを簡単に作れるようにするjQueryプラグインです。

Highlight Search Terms (jQueryプラグイン)

Highlight Search TermsはGoogleやYahoo!、Bing等の検索エンジンからアクセスした訪問者へリファラーに含まれるキーワードをハイライトしてあげるjQueryプラグインです。

Google Web Fonts Preview

Google Web Fonts Previewは任意の文字列とCSSでGoogle Web Fontsで公開されているフォントをプレビューするWebツールです。

140 Colors Found

140 Colors FoundはCSSで利用できる140種類の色キーワードをランダムに表示するWebページです。

Better Smart Keywords

Better Smart Keywordsはブラウザーについているプリフィックス型のスマートキーワード(ロケーションバーから直接検索する機能)より、サフィックス型の方が検索エンジンを切り替えやすく便利なのではないかという考えの元に作成したWebページ検索のフロントエンドです。

CSS Color Keywords

CSS Color KeywordsはCSSで利用できる140種類の色キーワードを全てプレビューするWebページです。

Test Font (jQueryプラグイン)

Test Fontは指定したフォントがインストールされているかどうかを簡単にチェックすることができるjQueryプラグインです。

&

&はGoogle Web Fontsで公開されている全てのフォントのアンパサンドをプレビューするWebページです。

Asamashi09

Asamashi09はJavaScriptのみで動作するAmazonアソシエイトのフロントエンド・ツールです。

Query YQL (jQueryプラグイン)

Query YQLはYQLへの複雑なクエリーをjQuery.getJSON()のようにシンプルに記述できるようにするjQueryプラグインです。

Table Filter (jQueryプラグイン)

Table FilterはHTMLの表にインクリメンタル検索で絞り込める入力フォームを追加するjQueryプラグインです。

xHashchange (jQueryプラグイン)

xHashchangeはhashchangeイベントをサポートしていない古いブラウザーへのみ同じインターフェイスの似たイベントを追加するjQueryプラグインです。
